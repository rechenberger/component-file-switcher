# component file switcher

## Features

Switches to corrosponding files with a diffrent extension.

For Example:
- open `app.component.ts`
- press `ctrl+cmd+s`
- pick `html`
- The extension opens `app.component.html`

## Extension Settings

Keybinding
``` json
{
  "command": "extension.switchTo",
  "key": "ctrl+cmd+s"
}
```
